# MailSystem

- 适用场景：延迟奖励、补偿、离线投递、附件领取。
- 常见触发词：邮件、附件、补偿、离线发放、批量发奖。
- 典型输入：邮件模板、收件人、附件奖励、过期时间、来源事件。
- 典型输出：邮件记录、附件领取状态、奖励流水。
- 配置关注：模板、过期时间、附件组、语言、批量发送范围。
- 协议关注：邮件列表、领取附件、重复领取、过期错误码。
- QA 关注：过期、重复领取、批量发送、背包满、断线重连。
- 常见组合：`RewardSystem`、`RankingSystem`、`ActivitySystem`。
- 不适用场景：实时同步且无需离线投递的奖励。
- 人工 Review：附件发奖、补偿和批量邮件必须 Review。
