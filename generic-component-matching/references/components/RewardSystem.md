# RewardSystem

- 适用场景：确定性奖励组、货币或道具发放、领取记录、奖励预览。
- 常见触发词：奖励、领取、发放、获得、补偿、首通、里程碑。
- 典型输入：奖励组 ID、领取条件、玩家 ID、来源事件、幂等键。
- 典型输出：奖励明细、背包变化、领取记录、奖励日志。
- 配置关注：奖励组、道具 ID、数量、绑定状态、领取次数、时间窗。
- 协议关注：奖励预览、领取响应、背包变化、错误码、重复领取。
- QA 关注：重复领取、背包满、配置缺失、断线重连、回滚和对账。
- 常见组合：`ActivitySystem`、`TaskCondition`、`MailSystem`、`RankingSystem`、`SettlementSystem`。
- 不适用场景：只展示奖励但不发放时，可只输出预览依赖。
- 人工 Review：任何真实资产发放都需要 Review。
