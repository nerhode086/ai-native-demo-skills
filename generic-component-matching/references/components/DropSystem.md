# DropSystem

- 适用场景：随机掉落、权重池、来源描述、掉落预览、掉率变更。
- 常见触发词：掉落、随机奖励、掉率、掉落池、权重、来源飘字。
- 典型输入：掉落池 ID、来源类型、玩家上下文、随机种子或权重条件。
- 典型输出：候选奖励、实际掉落、掉落来源、掉落日志。
- 配置关注：权重、保底、掉落池引用、活动加成、展示文本。
- 协议关注：掉落结果、来源字段、客户端展示所需字段。
- QA 关注：权重边界、空池、配置缺失、重复请求、弱网重试。
- 常见组合：`RewardSystem`、`InteractionObject`、`SettlementSystem`。
- 不适用场景：确定性奖励领取。
- 人工 Review：掉落影响经济、稀有奖励或付费价值时需要 Review。
