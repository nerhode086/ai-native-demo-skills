# InteractionObject

- 适用场景：场景物件、采集点、NPC 点击、宝箱、陷阱、传送门、世界交互对象。
- 常见触发词：采集、点击、打开、进入、触发、交互、场景对象。
- 典型输入：对象 ID、场景 ID、交互状态、可交互条件、玩家位置或距离。
- 典型输出：交互结果、对象状态变化、客户端表现、后续事件。
- 配置关注：对象配置、刷新规则、可交互条件、重复交互限制。
- 协议关注：交互请求、对象状态同步、错误码、断线重连状态恢复。
- QA 关注：重复点击、距离非法、对象已消失、配置缺失、断线重连。
- 常见组合：`RewardSystem`、`DropSystem`、`TaskCondition`、`ActivitySystem`。
- 不适用场景：纯 UI 按钮、无场景对象的后台结算。
- 人工 Review：交互触发资产变化或奖励发放时需要 Review。
