# ComposeSystem

- 适用场景：材料合成、道具合并、配方生成、碎片转换。
- 常见触发词：合成、制作、碎片、配方、兑换、转换。
- 典型输入：配方 ID、材料、目标道具、数量、玩家背包。
- 典型输出：材料扣除、目标道具、合成记录、失败原因。
- 配置关注：配方、消耗、产物、成功率、批量规则、限制。
- 协议关注：合成请求、幂等键、背包 delta、错误码。
- QA 关注：材料不足、批量合成、重复请求、背包满、配置缺失。
- 常见组合：`CostSystem`、`RewardSystem`。
- 不适用场景：无材料消耗的直接奖励。
- 人工 Review：稀有道具或付费材料转换需要 Review。
