# CostSystem

- 适用场景：货币、材料、体力、券、道具消耗。
- 常见触发词：消耗、扣除、花费、材料不足、购买、升级消耗。
- 典型输入：消耗项、数量、玩家资产、业务来源、幂等键。
- 典型输出：扣除结果、资产变化、失败原因、流水日志。
- 配置关注：消耗表、替代消耗、折扣、活动加成、上下限。
- 协议关注：预览消耗、确认消耗、资产 delta、失败错误码。
- QA 关注：不足、重复请求、并发扣除、断线、回滚。
- 常见组合：`LevelUp`、`ComposeSystem`、`ActivitySystem`、`RewardSystem`。
- 不适用场景：无资产变化的纯展示操作。
- 人工 Review：付费、稀有资产或不可逆消耗需要 Review。
