# LevelUp

- 适用场景：装备、坐骑、技能、伙伴、称号、天赋等持久成长。
- 常见触发词：升级、升星、突破、学习、成长、解锁、进阶。
- 典型输入：成长对象、当前等级、目标等级、条件、消耗。
- 典型输出：新等级、属性变化、解锁内容、战力变化、事件。
- 配置关注：等级表、消耗表、最大等级、属性增量、解锁规则。
- 协议关注：升级请求、幂等键、响应 delta、错误码、客户端刷新字段。
- QA 关注：材料不足、满级、重复点击、配置缺失、断线重连。
- 常见组合：`CostSystem`、`AttributeSystem`、`RewardSystem`、`TaskCondition`。
- 不适用场景：临时 buff、一次性邮件奖励、纯随机掉落。
- 人工 Review：涉及资产、属性、战力、付费来源时需要 Review。
