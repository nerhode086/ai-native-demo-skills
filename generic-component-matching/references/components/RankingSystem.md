# RankingSystem

- 适用场景：排行榜、排名奖励、快照结算、排名展示。
- 常见触发词：排行、榜单、名次、赛季、结算、排名奖励。
- 典型输入：排行类型、分数、玩家维度、赛季时间、排序规则。
- 典型输出：排名、快照、奖励区间、榜单展示数据。
- 配置关注：排序规则、同分规则、赛季时间、奖励区间、快照策略。
- 协议关注：榜单查询、我的排名、结算结果、奖励状态。
- QA 关注：同分、跨天/跨赛季、延迟数据、快照重建、发奖幂等。
- 常见组合：`RewardSystem`、`MailSystem`、`SettlementSystem`。
- 不适用场景：只展示个人历史数据且无排名关系。
- 人工 Review：排行结算和排行发奖必须 Review。
