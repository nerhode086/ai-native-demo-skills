# AttributeConvertRule

- 适用场景：一种属性转化为另一种属性，例如坐骑速度转主角速度、称号百分比加最终攻击。
- 常见触发词：转化、按比例、百分比、换算、来源属性影响目标属性。
- 典型输入：来源属性、目标属性、转化公式、生效条件。
- 典型输出：目标属性 delta、计算顺序、展示和日志。
- 配置关注：转化系数、上下限、叠加顺序、是否参与战力。
- 协议关注：来源值、转化结果、最终值、客户端展示字段。
- QA 关注：公式精度、边界值、来源变化、卸下/失效、战力重算。
- 常见组合：`AttributeSystem`、`LevelUp`。
- 不适用场景：同一属性的简单固定加成。
- 人工 Review：影响战力、结算或付费属性时需要 Review。
