# AttributeSystem

- 适用场景：属性发放、刷新、派生属性展示、战力参与、属性同步。
- 常见触发词：属性、力量、攻击、生命、速度、战力、加成。
- 典型输入：属性来源、目标对象、属性类型、修饰方式、生效条件。
- 典型输出：属性 delta、最终属性、战力变化、展示字段、日志。
- 配置关注：属性 ID、来源 ID、公式、系数、上下限、叠加规则。
- 协议关注：属性变化、最终值、展示依赖、日志追踪字段。
- QA 关注：装备/卸下、叠加、过期、重连、显示不一致、战力重算。
- 常见组合：`LevelUp`、`AttributeConvertRule`、`SettlementSystem`。
- 不适用场景：只发奖励且不影响属性。
- 人工 Review：影响战力、结算、排行、付费属性时需要 Review。
