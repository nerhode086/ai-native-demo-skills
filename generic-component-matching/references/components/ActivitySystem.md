# ActivitySystem

- 适用场景：限时规则、活动入口、日历门禁、活动奖励。
- 常见触发词：活动、限时、入口、日历、开启、关闭、领奖。
- 典型输入：活动 ID、活动状态、时间窗、玩家资格、活动配置。
- 典型输出：入口状态、活动状态、可领奖状态、活动进度。
- 配置关注：时间窗、开关、分组、奖励、任务、入口展示。
- 协议关注：活动状态、入口可见、领奖、错误码、旧客户端兼容。
- QA 关注：活动未开、已结束、跨时区、入口红点、重复领奖、灰度。
- 常见组合：`RewardSystem`、`TaskCondition`、`MailSystem`、`RankingSystem`。
- 不适用场景：永久功能且无活动门禁。
- 人工 Review：活动奖励、入口恢复、付费活动需要 Review。
